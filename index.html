How to test the voice AI

Option 1 — Real phone call: Dial +1 (930) 254-9264. The AI greets you with "Hello, you've reached the RTL Procurement Office." Try the example phrases below.

Option 2 — Verify the data directly: Each section below has a live tester that hits the same backend endpoint the voice AI uses. If the tester returns the right data, the AI can use it on a call.

Cold-start note: The backend sleeps after 15 min of no traffic on Render's free tier. The first request after sleep takes ~30 seconds to wake up. The voice AI tools are set to wait 45 seconds, but if you hear a long pause on a call, that's why.

4. Request Validation

tool: validate_request  ·  endpoint: POST /validate-request

What you can say on a call

  • "I need to submit a procurement request."
  • "I want to engage a new vendor for [project]."
  • "I'm trying to file a request for [thing]."

What the AI should say back

The AI gathers four required fields by asking follow-ups: budget number, milestones, costs, description (and optionally deadline). If anything's missing it asks for it. Only complete requests get routed — this is what prevents the 50% incomplete-request waste.
